# Learn Dashboard Capture

Use this guide to capture the new interactive dashboard screenshot.

## Target File Name

- `docs/images/learn-dashboard.png`

## Steps

1. Run the app locally:
   - `python app.py`
2. Open:
   - `http://127.0.0.1:5000/learn`
3. Set browser zoom to 100%.
4. Ensure the page shows:
   - track selector
   - checklist card
   - quick links card
5. Capture screenshot and save as:
   - `docs/images/learn-dashboard.png`

## Safety

- Hide personal tabs/bookmarks.
- Do not capture private tokens or profile menus.
